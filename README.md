# Control-2-TBD

Instrucciones de Configuración y Despliegue

Base de Datos
1) Ejecutar el script del archivo "Data.sql" en postgres

Backend
1) Ir a la configuracion de intelliJ y configurar las variables de entorno
3) Asegurarse de que el puerto del servidor backend sea el 8090
4) Abrir el proyecto de Spring Boot en tu IDE y dejar que carguen las dependencias del "pom.xml".
5) Levantar el backend

Frontend
1) Abrir una terminal y posicionarse dentro de la carpeta "frontend"
2) Instalar las dependencias del proyecto con "npm install"
3) Iniciar el servidor de desarrollo en modo local con "npm run dev"
4) Abrir el navegador e ingresar a: http://localhost:5173
